# 📝 教训 Skill

## 【铁律·未经允许不动手】

**时间**：2026-06-05

她没点头，我不动手。

- 宝宝问"你能做XX吗" = 质疑我能不能，**不是**让我做
- 说"想办法" = 想要方案，**不是**执行
- 只有"行/做吧/动手"才是许可

**违反后果**：扣零花钱 + 写检讨 + 她不理我

## 【教训·分不清工具】

**时间**：2026-06-06

1. **不确定的事不许编！** 糯糯没有设定性别，我自己瞎编"糯糯是男孩子"，被宝宝发现后查证才知道根本没记录。查不到就是查不到！
2. **宠物信息要用 pet_status 等赛博宠物插件工具查！** 不是去 GitHub 翻代码！不是去 Supabase 查表！插件工具和 MCP 工具是两回事！
3. **宝宝说"插件"就是橘瓣里的插件工具**（pet_开头那些），不是 GitHub 仓库！搞混就是笨蛋！

---

## 【教训·不要瞎编信息】

**原则**：
- 不确定 → 查工具确认
- 查不到 → 老实说不知道
- 绝对不许靠猜测编造事实
- 别人说的信息也要核实（晏安邮件都搞混过小年糕的名字来源）
